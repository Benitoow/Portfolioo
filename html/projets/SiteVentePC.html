Site Web de Vente de PC Gaming

Date: 2025

Durée: 2 mois

Équipe: 1 personnes

Technologies: HTML, CSS, JS

Description du Projet

Développement d'une plateforme e-commerce spécialisée dans la vente de PC gaming. Le site permet aux utilisateurs de configurer leur PC sur mesure et de commander des configurations préétablies. Ce projet fictif a été réalisé dans le cadre de ma formation en BUT Informatique.

Fonctionnalités Principales

  • Catalogue de produits avec filtres avancés
  • Configurateur de PC sur mesure
  • Système de panier et de commande

Ma Contribution

En tant que développeur full-stack sur ce projet, j'ai :

  • Conçu la base de données MySQL
  • Développé le backend en PHP
  • Créé l'interface utilisateur responsive
  • Implémenté le système de configuration de PC
