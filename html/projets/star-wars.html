Star Wars — Plateforme Web

Développement d'un site dynamique en PHP avec BDD JSON.

Voir le Site Live Fichiers Sources

Contexte & Thématique

Projet SAE 1.05 — BUT MMI. L'objectif de ce module était de concevoir un site web complet sur un sujet de notre choix. Grand passionné de science-fiction, j'ai sélectionné l'univers de Star Wars pour construire cette plateforme interactive.

Bien plus qu'un simple exercice d'intégration HTML/CSS, ce projet évaluait nos capacités à rendre un site dynamique côté serveur à l'aide de PHP, et à structurer/exploiter des données mockées sous format JSON.

  • Contexte : BUT MMI (Troyes)
  • Rôle : Développeur Web
  • Enjeu : Dynamisme & BDD JSON
  • Technologies : HTML5, CSS3, PHP 8, JSON

Défis Techniques & Fonctionnalités

Au-delà du design "sci-fi" soigné, l'intérêt principal du site réside dans son architecture backend légère et fonctionnelle.

Formulaire PHP

La page de Contact intègre un formulaire complexe traité en PHP natif, gérant les erreurs et les messages de succès via l'URL (GET).

Base JSON

L'ensemble du contenu data (personnages, vaisseaux) est stocké, parsé et généré dynamiquement en PHP depuis un fichier JSON.

Galerie Dynamique

Développement d'une galerie d'images organisée offrant une immersion visuelle optimale grâce aux inclusions PHP.
